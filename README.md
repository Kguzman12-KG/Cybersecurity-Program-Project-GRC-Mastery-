# Oscorp Cybersecurity Program Review  
### NIST CSF Assessment & 3-Year Risk Reduction Roadmap

> Disclaimer: This is a fictional cybersecurity consulting case study created for portfolio purposes.

---

## Overview

This project simulates an end-to-end cybersecurity consulting engagement for a research-driven organization (Oscorp) with high-value intellectual property.

The objective was to:
- Assess current cybersecurity posture  
- Identify critical control gaps  
- Design a 3-year cybersecurity roadmap  
- Translate cyber risk into financial impact  

---

## Problem

Oscorp lacked foundational cybersecurity controls, including:

- No multi-factor authentication (MFA)  
- Shared administrative credentials  
- No data classification or DLP  
- No centralized monitoring (SIEM)  
- Weak vulnerability management  

This created significant exposure to:
- Credential compromise  
- Data exfiltration  
- Intellectual property loss  

---

## Key Findings

- Breach likelihood: **25%–35% annually**  
- Financial impact: **$50M–$500M**  
- Annual risk exposure: **$15M–$150M**  

---

## 3-Year Roadmap

**Year 1 — Risk Containment**
- MFA, DLP, SIEM, vulnerability management  

**Year 2 — Maturity**
- PAM, CMDB, TPRM, incident response  

**Year 3 — Optimization**
- Insider threat detection, continuous monitoring, Zero Trust  

---

## Financial Impact & ROI

- Risk reduced to: **$5.5M–$55M annually**  
- Annual reduction: **$9.5M–$95M**  
- Investment: **$0.75M–$1.5M**  
- ROI: **8x – 94x**  
- Payback: **< 12 months**
